Agenda De Contatos

O que é?

O projeto agenda de contatos é um site feito por Pedro Herique para os estudos, onde o usuário cria uma conta e tem acesso a uma agenda virtual para salvar seus contatos, podendo colocar nome, email e telefone; e melhor ainda... sem limite!

Caso tenha se interessado você pode criar sua conta clicando aqui ou se já possui o cadastro, logar aqui .

Técnologias utilizadas

Html

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas web. Ele permite adicionar estrutura, conteúdo e formatação a um documento eletrônico, como texto, imagens e links. HTML é uma tecnologia básica e fundamental para o desenvolvimento de sites na internet.

Css

Além de aplicar estilos básicos, como fontes e cores, o CSS também permite criar layouts complexos, animações, responsividade para diferentes tamanhos de tela e muito mais. Ele é uma tecnologia fundamental para o design de páginas web, permitindo aos desenvolvedores controlar de forma precisa como o conteúdo HTML será apresentado aos usuários.

Veja mais

JavaScript

JavaScript é uma linguagem de programação de script, dinâmica e orientada a objetos. Ele é amplamente utilizado na criação de páginas web interativas e aplicações web. Com JavaScript, é possível adicionar funcionalidades a páginas HTML, como formulários de validação, animações, gráficos dinâmicos, entre outros.

Veja mais

Java

Java é uma linguagem de programação de propósito geral, orientada a objetos e altamente portável. É amplamente utilizada para desenvolvimento de aplicações desktop, servidores, dispositivos móveis, jogos e muito mais. Java oferece inúmeras bibliotecas e ferramentas para desenvolvimento, além de ser uma das linguagens mais utilizadas no mundo empresarial devido à sua escalabilidade e segurança.

Veja mais

Mysql

Quando se fala de Java e MySQL juntos, significa que o Java está sendo utilizado para se conectar e manipular dados no banco de dados MySQL. Isso permite criar aplicações web dinâmicas e robustas, com capacidade de acesso a grandes quantidades de dados armazenados no MySQL. A combinação de Java e MySQL é amplamente utilizada em aplicações empresariais e é uma solução poderosa e confiável para o desenvolvimento de aplicações web.
